CSSリファレンス

CSS（Cascading Style Sheets）とは、HTMLで定義されたウェブサイトのサイズや色、レイアウトなどを設定するための言語です。背景の色や画像の設置、文字のフォントなどの幅広いデザインを定義する際に使用されます。

HTMLの基本構造

CSSの要素

          

            

              

                
プロパティ

                
<h1>が最も大きく、<h6>が最も小さい見出しです。

                

              

                
セレクタ

                
段落を表します。文章を区切るときに使います。

              

              

                
値

                
リンクを表します。他のページへのリンクや、同じページ内の場所へのリンクを作成します。

              

              

                
<img>

                
画像を表示します。src属性に画像のURLを指定します。

              

              

                
<ul>

                
順不同リストを作成します。

              

            

          

        


        

          
主要プロパティ

          

            

              

                
height

                
<h1>が最も大きく、<h6>が最も小さい見出しです。

              

              

                
width

                
段落を表します。文章を区切るときに使います。

              

              

                
color

                
リンクを表します。他のページへのリンクや、同じページ内の場所へのリンクを作成します。

              

              

                
background-coler

                
画像を表示します。src属性に画像のURLを指定します。

              

              

                
margin
