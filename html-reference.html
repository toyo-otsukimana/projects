HTMLリファレンス

HTML（HyperText Markup Language）は、ウェブページを作成するための言語です。ウェブページの構造を定義し、ブラウザがそれを表示する方法を指示します。HTMLは、タグと呼ばれる特別な記号でテキストを囲むことで、テキストの意味や位置を指定します。

HTMLの基本構造

HTML文書は、次のような基本的な構造を持っています。

<html>

                      使い方:　
                      <html></html>

                          </h1>
                        

                        

                          <h2>
                            
サンプル

                          </h2>
                        

                        

                          <h3>
                            
サンプル

                          </h3>
                        

                        

                          <h4>
                          
サンプル

                          </h4>
                        

                        

                          <h5>
                            
サンプル

                          </h5>
                        

                        

                          <h6>
                            
サンプル

                          </h6>
                        

                    

                  

                  

                    
<p>

                    
段落を表します。文章を区切るときに使います。

                      使い方:

サンプル

                        </p>

<a>

                    
リンクを表します。他のページへのリンクや、同じページ内の場所へのリンクを作成します。

                      使い方:　
                      <a>リンクだよ</a>


                      関連する属性

                      
    
                        
  • href: リンク先のURL
    • 
                        
  • target: リンク先の開き方 (_blank, _self, _parent, _top)
    • 
                        
  • rel: リンクの関係性 (nofollow, noopener, noreferrer)
    • 
                      


                      使用例:　
                      <a href="https://example.com" target="_blank" rel="noopener noreferrer">Visit Example</a>
                    

                  

                

              



                

                  
主要タグ( 画像 )

                  

                    

                      
<img>

                      
画像を表示します。src属性に画像のURLを指定します。

                        使い方:　
                        <img src="image.png(jpgn) alt="代替テキスト">


                        関連する属性

                        
    
                          
  • src: 画像のURL
    • 
                          
  • alt: 代替テキスト
    • 
                        


                        使用例

                        <img src="image.jpg" alt="Sample Image" width="600" height="400">
                      

                    

                    

                      
<figure>

                      
画像や図、イラストレーション、チャートなどをグループ化するために使用されます。
これにより、図やメディアが文書の中で意味的に関連付けられ、コンテンツの理解が容易になります。

                        使い方:　
                        <figure></figure>

主要タグ( フォーム )

                  

                    

                      
<form>

                      
フォームを作成

                        使い方:　
                        <form></form>


                        関連する属性

                        
    
                          
  • action: フォームの送信先URL
    • 
                          
  • method: フォームの送信方法 (GET, POST)
    • 
                        


                        使用例:　
                        <form action="/submit" method="POST"></form>
                      

                    

                    

                      
<input>

                      
入力フィールド

                        使い方:　
                        <input>


                        関連する属性

                        
    
                          
  • type: 入力フィールドの種類 (text, password, submit, checkbox, radio, fileなど)
    • 
                          
  • name: フィールド名
    • 
                          
  • value: 初期値
    • 
                          
  • placeholder: 入力のヒント
    • 
                          
  • required: 入力必須
    • 
                        


                        使用例:　
                        <input type="text" name="username" placeholder="Enter your username" required>
                      

                    

                    

                      
<textarea>

                      
テキストエリア

                        使い方:　
                        <textarea></textarea>


                        関連する属性

                        
    
                          
  • type: 入力フィールドの種類 (text, password, submit, checkbox, radio, fileなど)
    • 
                          
  • name: フィールド名
    • 
                          
  • rows: 行数
    • 
                          
  • cols: 列数
    • 
                          
  • placeholder: 入力のヒント
    • 
                        


                        使用例:　
                        <textarea name="message" rows="4" cols="50" placeholder="Enter your message"></textarea>

  • value: 初期値

属性

                

                  

                    
id

                    
id: 要素の一意の識別子を指定します。

                      使い方:　
                      <div id="main-content"></div>
                    

                  

                  

                    
class

                    
要素に一つまたは複数のクラス名を指定します。

                      使い方:　
                      <p class="text-bold text-large"></p>
                    

                  

                  

                    
style

                    
インラインスタイルを指定します。

                      使い方:　
                      <h1 style="color: blue; font-size: 24px;"></h1>
                    

                  

                  

                    
title

                    
要素にツールチップを表示します。

                      使い方:　
                      <abbr title="HyperText Markup Language">HTML</abbr>
                    

                  

                  

                    
data-*

                    
カスタムデータ属性。JavaScriptでアクセス可能です。

                      使い方:　
                      <div data-user-id="12345"></div>
