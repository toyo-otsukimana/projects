HTMLリファレンス

HTML（HyperText Markup Language）は、ウェブページを作成するための言語です。ウェブページの構造を定義し、ブラウザがそれを表示する方法を指示します。HTMLは、タグと呼ばれる特別な記号でテキストを囲むことで、テキストの意味や位置を指定します。

HTMLの基本構造

HTML文書は、次のような基本的な構造を持っています。

よく使われるHTML要素

            

              

                
<html>

                  使い方:

                  
<html></html>

                

              

              

                
<head>

                

                  メタデータを含むセクション


                  使い方:

                  
<head></head>

                

              

              

                
<title>

                

                  ドキュメントのタイトル


                  使い方:

                  
<title></title>

                

              

              

                
<meta>

                

                  メタデータ


                  使い方:

                  
<meta>　(閉じタグ不要)

                  使い方:

                  
<link>　(閉じタグ不要)

HTMLの主要タグ

            

              

                
<h1>〜<h6>

                

                  <h1>が最も大きく、<h6>が最も小さい見出しです。


                  使い方:　

                    

                      
<h1> サンプル </h1>

                      
<h2> サンプル </h2>

                      
<h3> サンプル </h3>

                      
<h4> サンプル </h4>

                      
<h5> サンプル </h5>

                      
<h6> サンプル </h6>

                    

                

              

              

                
<p>

                
段落を表します。文章を区切るときに使います。


                  使い方:　

                  

                      
<p> サンプル1 </p>

                      
<p> サンプル2 </p>

                  

                

              

              

                
<a>

                
リンクを表します。他のページへのリンクや、同じページ内の場所へのリンクを作成します。


                  使い方:

                  
<a></a>



                  関連する属性

                  
    
                    
  • href: リンク先のURL
    • 
                    
  • target: リンク先の開き方 (_blank, _self, _parent, _top)
    • 
                    
  • rel: リンクの関係性 (nofollow, noopener, noreferrer)
    • 
                  


                  使用例:

                  

                    <a href="https://example.com" target="_blank" rel="noopener noreferrer">
　Visit Example
</a>
                  

                

              

            

          



            

              
主要タグ( 画像 )

              

                

                  
<img>

                  
画像を表示します。src属性に画像のURLを指定します。


                    使い方:

                    

                      <img src="image.png(jpgn) alt="代替テキスト">
                    



                    関連する属性

                    
    
                      
  • src: 画像のURL
    • 
                      
  • alt: 代替テキスト
    • 
                    


                    使用例

                    

                      <img src="image.jpg" alt="Sample Image" width="600" height="400">
                    

                  

                

                

                  
<figure>

                  
画像や図、イラストレーション、チャートなどをグループ化するために使用されます。
これにより、図やメディアが文書の中で意味的に関連付けられ、コンテンツの理解が容易になります。


                    使い方:

                    

                      <figure></figure>

主要タグ( フォーム )

              

                

                  
<form>

                  
フォームを作成


                    使い方:

                    

                    <form></form>
                    



                    関連する属性

                    
    
                      
  • action: フォームの送信先URL
    • 
                      
  • method: フォームの送信方法 (GET, POST)
    • 
                    


                    使用例:

                    

                    <form action="/submit" method="POST"></form>
                    

                  

                

                

                  
<input>

                  
HTMLで使用される要素の一つで、ユーザーがウェブページ上でテキストやデータを入力するためのコントロールを提供します。主にフォーム（form）要素内で使用され、ユーザーがテキストボックスやラジオボタン、チェックボックス、ドロップダウンリストなどの形式で情報を入力することを可能にします。


                    使い方:

                    

                      <input>
                    



                    関連する属性

                    
    
                      
  • type: 入力フィールドの種類 (text, password, submit, checkbox, radio, fileなど)
    • 
                      
  • name: フィールド名
    • 
                      
  • value: 初期値
    • 
                      
  • placeholder: 入力のヒント
    • 
                      
  • required: 入力必須
    • 
                    


                    使用例:

                    

                      <input type="text" name="username" placeholder="Enter your username" required>
                    

                  

                

                

                  
<textarea>

                  
HTMLで使用される要素の一つで、ユーザーが複数行のテキストを入力できるようにするためのコントロールです。主にフォームの一部として使用され、ウェブページ上でテキストの入力領域を提供します。


                    使い方:

                    

                      <textarea></textarea>
                    



                    関連する属性

                    
    
                      
  • type: 入力フィールドの種類 (text, password, submit, checkbox, radio, fileなど)
    • 
                      
  • name: フィールド名
    • 
                      
  • rows: 行数
    • 
                      
  • cols: 列数
    • 
                      
  • placeholder: 入力のヒント
    • 
                    


                    使用例:

                    

                      <textarea name="message" rows="4" cols="50" placeholder="Enter your message"></textarea>
                    

                  

                

                

                  
<button>

                  
HTMLでユーザーがクリックして何らかのアクションを起こすことができるボタンを定義するための要素です。通常、フォームの送信やJavaScriptの関数呼び出しなど、様々な操作に使用されます。


                    使い方:

                    

                      <button></button>
                    



                    関連する属性

                    
    
                      
  • type: ボタンの種類 (button, submit, reset)
    • 
                      
  • name: フィールド名
    • 
                      
  • value: 初期値
    • 
                    


                    使用例:

                    

                      <button type="submit">Submit</button>
                    

                  

                

                

                  
<label>

                  
HTMLで使用されるタグの一つです。このタグは、フォームコントロール（input要素やtextarea要素など）のラベルを定義するために使われます。具体的には、フォームの各入力項目の説明や目的をユーザーに提示するために利用されます。


                    使い方:

                    

                      <label></label>
                    

                  

                

                

                  
<select>

                  

                    要素は、HTMLで使用されるフォーム部品の一種です。通常、ドロップダウンリストとして知られており、ユーザーがリストから1つ以上の選択肢を選ぶことができます。


                    <select>要素は、<option>要素の子要素として使用され、各<option>要素は選択肢のテキストを表示し、value属性を通じて送信される値を定義します。ユーザーが選択した値は、フォーム送信時にサーバーに送信されるため、フォーム処理やデータ収集に便利です。


                    使い方:

                    

                      <select></select>
                    

                  

                

                

                  
<option>

                  
選択肢や選択の自由を意味します。特定の状況やコンテキストにおいて、複数の選択肢の中から1つを選ぶことができることを指します。


                    使い方:

                    

                      <option></option>
                    

                  

                

              

            



            

              
主要タグ( テーブル )

              

                

                  
<table>

                  
HTMLで使用されるタグで、表を作成するために利用されます。このタグを使用することで、複数の行と列からなる表を作成し、データを整理して表示することができます。


                    使い方:

                    

                      <table></table>
                    



                    関連する属性

                    
    
                      
  • border: テーブルの枠線の幅
    • 
                    


                    使用例:

                    

                      <table border="1"></table>
                    

                  

                

                

                  
<tr>

                  
HTMLで使用されるタグで、"table row"（表の行）の意味を持ちます。このタグは、表（テーブル）の中で1つの行を定義するために使用されます。通常、<table>タグの中に配置され、その中に<td>（table data、表のデータ）または<th>（table header、表の見出し）というタグが入っています。


                    使い方:

                    

                      <tr></tr>
                    



                    関連する属性

                    
    
                      
  • rowspan: 行の結合
    • 
                    


                    使用例:

                    

                      <tr rowspan="2"></tr>
                    

                  

                

                

                  
<td>

                  
はHTMLで使用されるタグで、"table data"（表のデータ）を意味します。具体的には、HTMLの表（<table>要素）内で使用され、各セル（セル）にデータを表示するために使用されます。


                    使い方:

                    

                      <td></td>
                    



                    関連する属性

                    
    
                      
  • colspan: 列の結合
    • 
                    


                    使用例:

                    

                      <tr colspan="2"></tr>
                    

                  

                

                

                  
<th>

                  
HTMLで使用されるタグで、表（テーブル）の見出し行を定義するために使います。具体的には以下のような役割があります。

                    
    
                      
  1. 表の見出し行の定義: <th>タグは通常、表の最初の行または列に配置され、そのセルが表のどの部分に属するかを示します。例えば、列の見出しや行の見出しを指定するために使用されます。
    2. 
                      
  2. スタイルと意味付け: <th>タグには、デフォルトで太字のテキストスタイルが適用される場合がありますが、CSS（Cascading Style Sheets）を使用してさらにスタイルをカスタマイズすることができます。
    3. 
                    

                    使い方:

                    

                      <th></th>

主要タグ( マルチメディア )

              

                

                  
<audio>

                  
HTMLで音声ファイルを埋め込むために使用されます。このタグを使うと、ウェブページ上で音声を再生することができます。


                    使い方:

                    

                      <audio></audio>
                    



                    関連する属性

                    
    
                      
  • src: 音声ファイルのURL
    • 
                      
  • controls: コントロール表示
    • 
                      
  • autoplay: 音声を自動再生します。
    • 
                      
  • loop: 音声を繰り返し再生します。
    • 
                      
  • muted: 音声をミュートにします.
    • 
                    


                    使用例:

                    

                      <audio src="audio.mp3" controls></audio>
                    



                    Chromeでダウンロードを禁止にするには

                    Google Chromeではデフォルトで音声の「ダウンロード」ボタンが表示されます。

                    controlslist="nodownload"を指定すると、このダウンロードボタンを非表示にできます。

                    
<audio controls controlslist="nodownload" src="~"></audio>



                    ※基本的にautoplayは使うべきではありません。「静かな電車の中で意図せずスマートフォンから音が再生されてしまう」のような不快なユーザー体験につながる可能性があるため注意が必要です。

                    ※主要なモバイル端末のブラウザでは上記のような理由からautoplayに対応していません。
                  

                

                

                  
<video>

                  
HTMLで動画を埋め込むために使用されます。このタグを使うと、ウェブページ上で動画を再生することができます。


                    使い方:

                    

                      <video></video>
                    



                    関連する属性

                    
    
                      
  • src: 音声ファイルのURL
    • 
                      
  • controls: コントロール表示
    • 
                      
  • width: 幅
    • 
                      
  • height: 高さ
    • 
                      
  • autoplay: 動画を自動再生します。
    • 
                      
  • loop: 動画を繰り返し再生します。
    • 
                      
  • muted: 動画の音声をミュートにします。
    • 
                      
  • poster: 動画が再生される前に表示する画像のURLを指定します。
    • 
                    


                    使用例:

                    

                      <video src="video.mp4" controls width="600" height="400"></video>
                    



                    ※基本的にautoplayは使うべきではありません。「静かな電車の中で意図せずスマートフォンから音が再生されてしまう」のような不快なユーザー体験につながる可能性があるため注意が必要です。

                    ※主要なモバイル端末のブラウザでは上記のような理由からautoplayに対応していません。

属性

            

              

                
id

                
id: 要素の一意の識別子を指定します。


                  使い方:

                  

                    <div id="main-content"></div>
                  

                

              

              

                
class

                
要素に一つまたは複数のクラス名を指定します。


                  使い方:

                  

                    <p class="text-bold text-large"></p>
                  

                

              

              

                
style

                
インラインスタイルを指定します。


                  使い方:

                  

                    <h1 style="color: blue; font-size: 24px;"></h1>
                  

                

              

              

                
title

                
要素にツールチップを表示します。


                  使い方:

                  

                    <abbr title="HyperText Markup Language">HTML</abbr>
                  

                

              

              

                
data-*

                
カスタムデータ属性。JavaScriptでアクセス可能です。


                  使い方:

                  

                  <div data-user-id="12345"></div>
